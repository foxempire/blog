小众愚乐 2019年美国名校免费线上课程清单

2019-01-12作者:织梦猫来源：admin次阅读

如果你决定在2018年继续接受教育的话，那么足不出户就能有很多的学习机会。尽管网上拥有丰富的免费大学课程，但方便并不意味着你必须得在质量上妥协。为了让大家获取互联网上所提供的最好的免费教育，Quartz汇编了一份常春藤盟校免费课程清单(400门)。

八所常春藤联盟大学(哈佛大学、耶鲁大学、宾夕法尼亚大学、普林斯顿大学、哥伦比亚大学、达特茅斯学院、布朗大学及康奈尔大学)提供有约396门免费的在线课程。虽然其中许多都涉及传统科目，如市场营销、计算机科学和希腊神话，但也有不少独特的课程，包括《比特币与加密货币技术》与《记忆的伦理》等课程。

此外，许多课程都可由学习者自主调整进度，并会提供收费证书，而这笔费用可能会在50美元到100美元之间。这便意味着你所学习的课程在理论上并不免费，但仍然比前往哈佛大学现场听课要便宜很多。

在常春藤盟校今年提供的免费课程中，我们在下面列出了其中值得注意的25门：

1. 《比特币与加密货币技术》// 普林斯顿大学

2. 《人工智能(AI)》 // 哥伦比亚大学

3. 《动画与CGI动画》// 哥伦比亚大学

4. 《全球金融危机》// 耶鲁大学

5. 《众筹》 // 宾夕法尼亚大学

6. 《病毒式营销与如何制造传染性内容》 // 宾夕法尼亚大学

7. 《日常生活中的道德》 // 耶鲁大学

8. 《古希腊英雄》 // 哈佛大学

9. 《可视化日本(1850s-2030s)：西化、抗议、现代性》// 哈佛大学

10. 《美国资本主义：历史》// 康奈尔大学

11. 《科学与烹饪：从高级烹饪术到软物质科学(第一部分)》// 哈佛大学

12. 《权力与责任：跟超级英雄研究哲学》 // 哈佛大学

13. 《超级地球与生命》// 哈佛大学

14. 《女性创造历史：十个对象，多个故事》 // 哈佛大学

15. 《世界文学中的现代杰作》 // 哈佛大学

16. 《记忆的伦理》// 布朗大学

17. 《佛教与现代心理学》 // 普林斯顿大学

18. 《设计城市》 // 宾夕法尼亚大学

19. 《美国的成文宪法》 // 耶鲁大学

20. 《埃博拉的教训：预防下一次流行病》 // 哈佛大学

24. 《意大利歌剧导论》 // 达特茅斯学院
