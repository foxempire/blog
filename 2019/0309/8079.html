小众愚乐 世界各地的13个古怪路标

2019-03-09作者:织梦猫来源：admin次阅读

在异国他乡开车会非常具有挑战性。无论是司机座位有可能相反，或是按本国习惯开车有可能在逆行，还是路标可能令人感到迷惑，这些琐事总是会让人心烦意乱。就比如说，以下这13个世界各地的古怪路标，你能看懂几个呢？

1. 陡峭悬崖旁的路标 // 阿根廷

[点击查看图片]

不，阿根廷的汽车并不能漂浮起来。这个路标是要提醒司机控制速度，以面对即将到来的小山。

2. 美洲驼横穿路标// 玻利维亚

[点击查看图片]

当你在玻利维亚等南美国家开车时，应该时刻警惕横穿马路的美洲驼，不然小心它们对你吐口水。

3. 盲山路标 // 冰岛

[点击查看图片]

听好了！那个感叹号只是为了提醒你注意即将到来的blindhæð，或者说是“盲山”，也就是你看不到对面是否有人正开过来的小凸坡。

4. 刺猬路标 // 英国

[点击查看图片]

如果你在英国开车还敢无视这个路标，那你最好确保自己拥有最坚固的轮胎。这个路标是在警告驾驶者附近有非常可爱却满身长钉的刺猬。否则的话，你在驾驶时最好额外小心，以避免碾到这些小家伙。

5. 雪橇横穿路标 // 格陵兰岛

[点击查看图片]

在一个道路很少、冰却很多的国家，狗拉雪橇是最好的出行方式之一。格陵兰岛的这个路标就是在提醒司机前方可能有雪橇横穿。

6. 羚羊横穿路标 // 南非

[点击查看图片]

我们曾在新闻中看到羚羊会攻击无人机，但我们不知道它是否也会攻击我们的汽车。为了安全起见，请注意几个南部非洲国家所设置的这些路标。

7. 禁行路标 // 以色列

[点击查看图片]

虽然看上去可能像是某个棋盘游戏的着陆区，但这实际上是一种以色列路标，让司机们知道该区域禁止以什么样的交通方式通行。

8. 死胡同路标 // 德国

[点击查看图片]

这个德国路标并非表示一个跳杆的顶部，或者是附近有自行车打气筒。在这个路标上，白线顶部的红线实际上代表一条死胡同。

9. 禁止超车路标 // 俄罗斯

[点击查看图片]

不，红车并非拥有优先通行权。这个俄罗斯路标是在告诉司机这条道路是双向道路，不同的颜色意味着禁止超车或通行。

10. 袋鼠横穿路标 // 澳大利亚

[点击查看图片]

澳大利亚到处都是想要杀死你的可爱野生动物，所以当你在这个国家的高速公路上行驶时要额外小心。可能会有一只嗜杀成性的袋鼠盯上你的本田。

11. 战场路标 // 英国

[点击查看图片]

如果你认为这个路标表示左转进入战场，那么你猜对了，不过只是代表著名的战场遗址。就像美国一样，英国的棕色路标表示旅游景点和历史遗迹。

12. 沿海路标 // 爱尔兰

[点击查看图片]

没错，这个路标看上去类似于一头大波浪，不过这个爱尔兰路标代表野生大西洋之路：一条全长1550英里、沿岛上风景秀丽的西海岸而建的步道。

13.骆驼横穿路标 // 以色列

[点击查看图片]

在美国，你可能会看到表示前方有减速带的路标。而在以色列，你会发现表示前方有减速骆驼的路标。
