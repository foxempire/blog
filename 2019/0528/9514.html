小众愚乐 新手养猫小技巧：教你三招解决猫咪乱尿问题！

2019-05-28作者:织梦猫来源：admin次阅读

很多铲屎官都被家里的主子乱尿问题所困扰，但又不知道如何解决，或者按照某些方法做了之后并没有什么效果，怎么办？没关系，今天小编出马教你三招搞定猫咪乱尿问题！

具体情况要具体解决，要想解决猫咪乱尿的问题，我们首先要找到猫咪乱尿的原因。那么猫咪乱尿都有哪些原因呢？通常是以下三种原因引起的，请各位铲屎官对症下药。

一、疾病问题

原因：主子一直都好好的，环境也没什么变化，但是突然不上厕所，选择拉在外面，那么可能是由疾病引起的，疾病有很多，在这里就不一一举例了。

治疗办法：建议看医生。如果是老猫，可能是行动不便，建议换个侧入式的猫砂盆。

二、心理压力

1.家里突然新增或减少人类成员或其他动物。

2. 作息大幅度改变或者疼痛不适。

3. 邻居的猫或者室外的野猫在游荡。

4. 家里主子们的关系改变或者不习惯新的猫咪，相互之间有攻击行为。

以上原因都是引起猫咪心理压力，导致的乱尿原因。具体表现为，尿在靠近室外的窗户、地脚线、墙壁之类的位置，大概是受到了外部的威胁，一般是由其他的猫咪引起的。尿在室内的家具或者门上，那就是和家里的其他猫咪有矛盾。尿在衣服、被子、椅子、键盘等地，那就是对物主的人类存在焦虑心态。

治疗方法：消除猫尿气味，把猫砂盆放在乱尿的旁边，提醒猫咪厕所就在旁边，进行行为指导。

三、猫砂盆厌恶

原因：猫砂盆犹犹豫豫，探头探脑看半天，只把前爪放在厕所里，而不是直接跳进去，拉完立刻跑出来，或者是靠近闻一闻就走，忍不住之后直接拉在厕所附近，那么基本可以确定是以下几种原因造成的。

1. 很少清理猫砂盆

2. 使用化学物品，猫咪不舒服

3. 猫咪不喜欢猫砂盆

4. 摆放位置不对

5. 猫砂问题

治疗办法：

1. 勤清理猫砂盆

2. 换个猫砂盆，最好是猫咪身长的1.5倍左右

3. 把猫砂盆放在一个安静的地方，不要和饭碗放在一起

4. 换猫砂
