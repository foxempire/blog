小众愚乐 云游戏将改变游戏界

2018-06-18作者:织梦猫来源：admin次阅读

游戏机生产商长久以来都处于游戏宇宙的中心位置，但随着即将到来的云游戏方式，他们需要尽快适应一系列迅速增加的游戏方式。

本周二在洛杉矶举办的E3电子娱乐展的展台上，微软、索尼和任天堂出品的竞争主机Xbox、PlayStation和Switch仍分别是游戏硬件界的明星产品。

但这个重要的游戏产业集会的重点是游戏本身，以及可以在一系列包括智能手机和电脑等不同设备上通过网络云端作为服务器进行游戏的理念。

根据IHS Markit游戏技术研究总监Piers Harding-Rolls所说，今年的E3盛会“举办在游戏主机市场有所下滑的背景下”。

“随着游戏市场越来越数字化并且服务越来越重要，游戏机生产商和游戏发行商开始着手制定一个长期的策略，包括新增订阅云游戏服务，”Harding-Rolls告诉AFP说。

根据分析，那些对云游戏感兴趣的公司将其视作获得更广泛的游戏玩家的一种方式，他们会把云游戏添加到游戏机商业中，而不是让二者分离。

Harding-Rolls说：“在目前和可预见的未来中，主机仍占有一席之地。”

但是这个竞争的天平将会从游戏机市场中占主导地位的PlayStation倾斜向在云游戏上更有优势的Xbox，他解释说。

微软已经打造好了一个强力的主机平台来搭载网络云的计算，使得这项服务可以成为他们业务成功的一部分。

Xbox的团队主管Phil Spencer在E3上的一个简报中说道：“在未来的游戏上，我们承诺并且会利用我们在微软的全部资源来向玩家呈献。”

Redmond是一家扎根于华盛顿的资深科技公司，也在机器学习上投入了巨资，他们用Xbox上最火爆的游戏《光环》中的人工智能角色将其命名为Cortana数码助手。

Spencer同时表示微软正着手于一项可以把主机品质的游戏传输到连接互联网的设备中的云服务。

“如果你认同最终未来的游戏消费是通过云游戏服务实现的，那么那些在云技术上处于强力地位的公司很可能在转变中盈利，”Harding-Rolls说。“在这一方面，当微软想要打造一个盈利的业务时，他们的云技术分部就赋予了微软这样一个天然优势。”

索尼和微软都已经放话说已经着手开发新一代的主机，但是新主机的参数还没有曝光。PS4目前仍统治了这一代主机的市场，销量远超Xbox One。

据Harding-Rolls所说，但如果电子游戏界转向微软的强项，那么微软在游戏界的新对手将会是云技术和AI界的大佬们，比如说亚马逊、谷歌、腾讯以及阿里巴巴。

据电子软件协会(Electronic Software Association)(个人觉得这里应该是Entertainment Software Association)的执行会长Michael Gallagher所说，由于三大游戏主机竞争“十分健康”，电子游戏产业把云游戏当做一次前所未有的最大投资。

ESA的贸易集团组织了E3展，来自大约100个国家的6万名业内人士和游戏玩家前来参加了这个为期3天的游戏盛会。

据ESA所述，去年电子游戏产业的全球收入合计差不多1160亿美元。

游戏机生产商也从移动游戏中学到了不少，他们搭建了可供玩家反馈的在线社区、增加了直播游戏、订阅服务以及可以花钱购买到的数字内容，比如说游戏角色的跳舞动作或者炫酷的时装。

Gallagher说：“移动端游戏一直是产业中非常出色的一个增长点。”

主机游戏中的巨头Bethesda公司也不加掩饰的将主机上热卖的游戏《辐射》和《上古卷轴》推出了移动平台版本，供玩家在智能手机或智能平板上游玩。

当提起《上古卷轴：刀锋》时Gallagher说：“我都等不及了。”这款游戏将在今年晚些时候登陆iPhone和安卓平台。

另外一个信号表明主机品质的游戏将会登陆移动设备是在E3的展台上，电信业巨头AT&T炫耀了一番超高速的5G无线数据传输能力。

超高速的网络服务是丰富又流畅的游戏体验的关键所在。

“计算能力、储存能力和网速的飞速发展将会把逼真的、系统的、人口密集和持久的游戏世界带到每一个游戏平台，” 法国最强大的游戏公司土豆种植基地育碧公司的董事长Yves Guillemot说道。
