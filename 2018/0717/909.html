# TIFU 是reddit上的一个分享自己糗事的板块，全称是Today I Fuck Up

# 如无意外今后将以10篇FML和一到两则TIFU小故事为一期

credit: 蛋友youhehe0

1. Anonymous

今天，我有一个千载难逢的机会可以见到汉密尔顿本人。但是我朋友一直在大声抱怨以及问我，”你来这就只是为了见这个人？“然后我们就被赶出去了。FML

# 汉密尔顿指的应该是F1冠军车手刘易斯-汉密尔顿。

2. Anonymous

今天，和我交往了四个月的男朋友说他在我的眼睛里看到了对他的感情。他有点抓狂因为这让事情变得复杂了起来。FML

3. Anonymous

今天，我92岁的曾祖父不顾他有严重的关节炎兴奋的跑下楼梯。他告诉我曾祖母他隔了10年后又一次勃起了。我曾祖母当即让我离开，好让他们来一发。FML

4. Tall Oaf

今天，和我分手2月缺却依然保持良好关系的前女友在Instagram上发了一张和他新男友的合照，标题是“爱你哦，宝贝。这4个月是我最幸福的4个月。”我数学不好，但是4应该是比2大的。FML

5. Piggirl

今天，是我们结婚12周年的纪念日的第二天。我在我丈夫口袋里找到了一张衣原体检测的收据，但是他打死不认他有了外遇。我们都两年没做过了好吗？FML

# 衣原体感染是一种常见的性传播疾病

6. CanntPoop

今天，我刚完告诉我妻子当你感到口渴的时候就一定要喝水这种话完全是无稽之谈。我的肚子突然就疼了起来。医院给的诊断结果是因脱水而导致的便秘，所以我的老老实实的接受静脉输液。医药费一共是2000刀。FML

7. catcat

今天，我第一次穿丁字裤。这条丁字裤是我几个礼拜之前买的，但是我现在才有足够的勇气穿上它。然后我就出门了，但是丁字裤卡在我臀沟导致了我菊部破碎，现在我不得不往屁股那塞纸巾，不然血止不住。FML

8. Anonymous

今天，我错过了公共汽车。所以我不得不忍受着45℃的高温赶往地铁站，在去地铁站的途中我不小心踩到狗屎摔倒在楼梯上，我的眼镜和左前牙全都摔坏了。当我准备站起来的时候，我的包卡在了栏杆里然后被我扯坏了。FML

10. Eheath12Jheath14

今天，我和我男朋友迎来了我们的第一个孩子 。当我生完孩子后，他说他要先回趟家确保房子里没有什么不适合婴儿的物品。几个小时后，我发现我们两的脸书共同好友在当地酒吧发了张照片，照片里有他。FML

o 祸从口出

我有个奇怪的习惯就是不管什么时候别人问我有没有在认真听他说话或者把注意力放在他身上的时候，我都会回一句“我还活着”。一个小时前我的一个好朋友她爹去世了，死于心脏病。我离她住的地方有上千英里远，所以我只能在电话里给她安慰。我有点吃惊因为她爹还很年轻而且这事发生的太突然了，所以，我在安慰她的时候没说太多话。她在电话里谈论了她将如何面对没有父亲的生活以及她将会十分想念她的父亲。我一言不发静静的听着她宣泄心里的悲伤。她以为我挂了于是我问了一句“你还在听吗？”。我顺嘴就是一句“对，我还活着”。:( 她听到后沉默不语，我真想找个地洞钻进去死了算了。
